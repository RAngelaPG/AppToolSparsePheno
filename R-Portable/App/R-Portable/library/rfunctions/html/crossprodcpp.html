crossprodcpp {rfunctions}R Documentation

Compute X'X or X'WX where W is diagonal weight matrix

Description

Compute X'X or X'WX where W is diagonal weight matrix

Usage

crossprodcpp(x, w = NULL)

Arguments

x

matrix input

w

numeric vector. If specified, X'diag(w)X will be computed

Value

A symmetric matrix X'X

Examples

n.obs <- 1e5
n.vars <- 150

x <- matrix(rnorm(n.obs * n.vars), n.obs, n.vars)

## compute X'X
xpx <- crossprodcpp(x)

weights <- runif(n.obs)

## compute X'WX
xpwx <- crossprodcpp(x, weights)
