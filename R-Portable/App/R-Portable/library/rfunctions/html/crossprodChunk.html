Compute X'X in chunks

Description

Compute X'X in chunks

Usage

crossprodChunk(X, row.chunk = NULL, sparse = FALSE)

Arguments

X

matrix input

row.chunk

integer. Number of rows per chunk. Specify NULL for no chunking

sparse

Boolean value to specify if matrix X is sparse

Value

A symmetric matrix X'X

Examples

crossprodChunk(matrix(rnorm(1e4 * 1e2), 1e4, 1e2), row.chunk = 500, FALSE)
