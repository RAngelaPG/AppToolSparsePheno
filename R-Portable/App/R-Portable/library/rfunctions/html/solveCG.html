solveCG {rfunctions}R Documentation

Solve the linear system Ax = b using conjugate gradient

Description

Solve the linear system Ax = b using conjugate gradient

Usage

solveCG(A, b, maxit = 500L, tol = 1e-05, preconditioner = c("none",
  "incompleteLU", "scaling"))

Arguments

A

square matrix input

b

numeric vector.

maxit

integer. Maximum number of iterations

tol

tolerance level

Value

list

Examples

n <- 2000
p <- 10
lambda <- 0.1

set.seed(100)
x <- matrix(rnorm(n * p), ncol = p)
y <- x %*% rnorm(p) + rnorm(p)
b <- y
A <- tcrossprod(x) + lambda * diag(n)

system.time(alpha.true <- solve(A, b))
system.time(alpha.cg <- solveCG(A, b))

max(abs(alpha.cg$x - alpha.true))
