distribution function

Description

distribution function

Usage

boost.pnorm(q, mean = 0, sd = 1, lower.tail = TRUE)

Arguments

q

vector of quantiles

mean

scalar mean (vector will be implemented)

sd

scalar standard deviation (vector will be implemented)

lower.tail

logical; if TRUE (default), probabilities are P[X <= x] otherwise, P[X > x].

Value

the distribution function

Examples

library(rfunctions)
vals <- rnorm(100, mean = 4, sd = 2)
boost.pnorm(vals, mean = 4, sd = 2, lower.tail = FALSE)
