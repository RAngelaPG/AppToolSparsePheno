simSparseMatrix {rfunctions}R Documentation

Simulate a matrix with very few nonzero entries

Description

Simulate a matrix with very few nonzero entries

Usage

simSparseMatrix(sparsity, dim, boolean = FALSE, slam = FALSE)

Arguments

sparsity

numeric value between 0 and 1 indicating level of sparseness

dim

numeric vector of length 2 indicating matrix dimensions

boolean

TRUE/FALSE. if true, matrix will only take values 0 and 1

slam

if TRUE, will return matrix in simple_triplet_matrix form. If FALSE, will return SparseMatrix object

Value

SparseMatrix or simple_triplet_matrix

Examples

n.obs <- 1e5
n.vars <- 1e3

#simulate a very sparse matrix (this matrix has many zeros and few ones)
x.sparse <- simSparseMatrix(sparsity = 0.99, dim = c(n.obs, n.vars), boolean = T)
