Compute X[,indices]'y

Description

Compute X[,indices]'y

Usage

sliced.crossprod(x, y, indices)

Arguments

x

matrix input

y

numeric vector.

indices

integer vector of X columns to use

Value

x[,indices]'y

Examples

n.obs <- 1e4
n.vars <- 2000
n.idx <- 500

x <- matrix(rnorm(n.obs * n.vars), n.obs, n.vars)
y <- rnorm(n.obs)
idx <- sample.int(n.vars, n.idx)

## compute x[,idx]'y
xpy <- sliced.crossprod(x, y, idx)

xpy2 <- crossprod(x[,idx], y)

all.equal(xpy, xpy2)
