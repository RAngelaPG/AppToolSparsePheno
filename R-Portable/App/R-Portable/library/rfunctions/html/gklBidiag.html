gklBidiag {rfunctions}R Documentation

Compute Largest Singular Value of Matrix x

Description

Compute Largest Singular Value of Matrix x

Usage

gklBidiag(x, v, maxit, reorthog = 0, upper.bound.prob = NULL)

Arguments

x

matrix input

v

numeric vector. Initialize GKL bidiagonalization with random vector on unit sphere

maxit

integer. Maximum number of Lanczos iterations

reorthog

Takes values in 0:2. 0 for no reorthogonalization, 1 for reorthogonalization of V vectors (slower, more accurate), 2 for reorthogonalization of V and U vectors (slowest and most memory, most accurate. Not implemented yet)

upper.bound.prob

upper bound probability for the largest singular value

Value

list

References

Hochstenbach (2013) Probabilistic upper bounds for the matrix two-norm, http://link.springer.com/article/10.1007/s10915-013-9716-x Journal of Scientific Computing, Vol. 57(3), Dec. 2013

Examples

n.obs <- 1e5
n.vars <- 150

x <- matrix(rnorm(n.obs * n.vars), n.obs, n.vars)

## compute largest singular value of x
lanczos <- gklBidiag(x, runif(ncol(x)), 10L, reorth = 0, upper.bound.prob = 0.99)
str(lanczos)
